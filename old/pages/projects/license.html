License API
VIEW CODE VIEW PROJECT
This project was a fairly simple, yet very useful project. When monitizing desktop applications, I was faced with the challenge of creating a license based authentication system. This API was created in response with Gofiber as the REST API framework and MySQL as the database language. Additionally, because I primarily use Discord as a medium to interact with the users of my applications, I added Discord integration that allows for users to bind their Discord accounts to their licenses.
